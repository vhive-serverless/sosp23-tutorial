Tutorial Description

This tutorial provides introduces the vHive ecosystem, a full-stack open-source framework for serverless experimentation and innovation. The tutorial seeks to educate the community about serverless computing architecture and benchmarking methodology, and teach the researchers from the computer systems community to use vHive for their research. The tutorial includes a number of hands-on sessions on analyzing performance of serverless applications and systems in production and open-source serverless clouds as well as instrumenting and optimizing serverless infrastructure across the whole stack.
